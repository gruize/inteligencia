Misioneros vs. Caníbales
  • Descripción del juego:

Debemos pasar 3 caníbales y 3 misioneros de una orilla del río a la otra, mediante una barca que sólo tiene 2 plazas para ser ocupadas, teniendo en cuenta que hay que evitar que el/los caníbal/es devore a un/los misionero/s.

  • Operadores

Todos los operadores suponen como requisito que la barca este ocupada para poder moverse de una orilla a la otra.

  1. Cruza un caníbal desde la orilla izquierda.

  2. Cruza un caníbal desde la orilla derecha.

  3. Cruzan dos caníbales desde la orilla izquierda.

  4. Cruzan dos caníbales desde la orilla derecha.

  5. Cruza un misionero desde la orilla izquierda.

  6. Cruza un misionero desde la orilla derecha.

  7. Cruzan dos misioneros desde la orilla izquierda

  8. Cruzan dos misioneros desde la orilla derecha.

  • Algoritmo

Algoritmo DepthLimitedSearch o Profundidad limitada: Basada en la búsqueda primero en profundidad, pero fijando un límite l de profundidad, para evitar descender infinitamente. Éste límite permite desechar caminos en los que el estado objetivo es muy lejano del nodo inicial. En este caso, el límite de profundidad es 11.

  • Espacio de estados

Estado( Número de misioneros en la orilla izquierda, Número de caníbales en la orilla izquierda, Posición de la barca )
     Número de misioneros en la orilla izquierda: {0, 1, 2, 3}
     Número de caníbales en la orilla izquierda: {0, 1, 2, 3}
     Posición de la barca: True (Orilla izquierda) v False (Orilla derecha)

Estados inalcanzables:
     Estado( 1, 0, Izquierda )
     Estado( 1, 2, Izquierda )
     Estado( 1, 3, Izquierda )
     Estado( 2, 0, Izquierda )
     Estado( 2, 1, Izquierda )
     Estado( 2, 3, Izquierda )
     Estado( 1, 0, Derecha )
     Estado( 1, 2, Derecha )
     Estado( 1, 3, Derecha )
     Estado( 2, 0, Derecha )
     Estado( 2, 1, Derecha )
     Estado( 2, 3, Derecha )

Estado inicial:
     Estado( 3, 3, Izquierda)

Estado final:
     Estado( 0, 0, Derecha)
