8 - Reinas
  • Descripción del juego:

    El problema de las N - Reinas consiste en poner n reinas de ajedrez en un tablero n x n, de tal manera que ninguna de ellas pueda atacar a otra usando las movidas estándar de una reina de ajedrez.

  • Heurística

El objetivo del juego consiste en rellenar el tablero n x n con n reinas, por lo que al ir aumentando el número de reinas en el tablero, disminuye la medida heurística

h'(n) = Dimensión del tablero - Número de reinas colocadas en el tablero

h'(n) = 8 - Número de reinas colocadas

  • Operadores

  1. Colocar reina en la posición [ i , j ] donde i = { 1, 2, 3 } y j = { 1, 2, 3 }

  • Algoritmo

Algoritmo UniformCostSearch o Coste Uniforme: Los nodos se expanden por orden no decreciente de coste (camino de menor coste), implementándose con una cola de prioridad, donde los nodos con igual coste generan el uso del algoritmo de búsqueda en anchura.

 

  • Espacio de estados

Estado ( Tablero [ ] [ ] )

Estados prohibidos:
     Los estados prohibidos son todos aquellos en que 2 o más reinas pueden atacarse.

Estado inicial:
     Tablero vacío

Estado final:
     Tablero conteniendo n reinas sin que ninguna de ellas sea amenazada por otra.
